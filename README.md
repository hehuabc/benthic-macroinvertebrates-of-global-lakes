# benthic-macroinvertebrates-of-global-lakes
This is the R codes and raw files of our manuscript "Climate mediates the species-area relationship of benthic macroinvertebrates across freshwater shallow lakes" which were
submitted to Global Ecology and Biogeography. We open them for the editors and reviewers. 
These files will also be open accessed in Zenodo later if the paper is accepted.  
This project include five files. Zoobenthos species data of each studies were in species.zip file. 
The morphological and nutrient status of each studied lake were in env.zip file. 
Before run the codes, both zip files should be decompressed and placed in work place.
Codes were seperated in two R files. The first one showed how to calculate rarefied richness and evenness metrics using the species data. 
We also provided the results of these codes in "diversity.csv" file. 
The other one showed the analysis and plot process of ISAR using "diversity.csv" file.
