# benthic-macroinvertebrates-of-global-lakes
This is the R codes and data files of our manuscript "Climate mediates the species-area relationship of benthic macroinvertebrates across freshwater shallow lakes" which were submitted to Global Ecology and Biogeography. We open them for the editors and reviewers. 

These files will also be open accessed in Zenodo later if the paper is accepted. 

We did not provide the raw data as they are generating another paper. However, We provided the results of rarefied richness (Sn) and SPIE in "diversity.csv" file. We then showed the analysis and plot process of ISAR using "diversity.csv" file in R. Anyone who want to know the calculating process of Sn and Spie could go to Chase-Lab: https://github.com/chase-lab/ISAR_synthesis.
